NC-SSM Noise-Conditioned State Space Model

One-Page Technical Summary
Overview

NC-SSM is an ultra-compact keyword spotting architecture that achieves structural noise robustness through noise-conditioned SSM dynamics. It matches CNN accuracy with 5–10× fewer MACs and 5× lower latency on Cortex-M7, eliminating the need for separate denoising modules while maintaining sub-10ms response times.

Architecture
Audio → STFT → Mel → SNR → DualPCEN → SSM×2 → Classifier

Key innovation: 5 noise-conditioning mechanisms that modulate SSM dynamics (Δ, B, C) based on per-band SNR estimation, enabling mathematically provable noise attenuation without external denoisers.

Competitive Advantages
  • Structural noise robustness — no denoising module needed; robustness is built into SSM dynamics
  • 3× faster streaming — 350 ms detection latency vs 1,053 ms for CNN baselines
  • Minimal state memory — 720 bytes vs 196 KB CNN feature maps (272× reduction)
  • Zero-power silence — 0 MACs during silence intervals, enabling up to 10× battery life improvement
  • Provable guarantees — formal proof of noise attenuation properties via Lyapunov analysis
  • Hardware-friendly — fits entirely in on-chip SRAM; no external memory required
